# AI Examples

```c++
int x = 5; // integer variable x initialized with value 5
double y = 10.5; // double precision floating point variable y initialized with value 10.5
char z = 'A'; // character variable z initialized with value 'A'
bool flag = true; // boolean variable flag initialized with value true

// Multiple variables can be declared in a single declaration
int, float, char x1 = 5, y1 = 3.14, z1 = 'B';
```

Let's see an example in Java

```java
// Declare an integer variable named 'age'
int age = 25;

// Declare a string variable named 'name'
String name = "John Doe";

// Declare a boolean variable named 'isMarried'
boolean isMarried = true;
```
